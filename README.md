# 📅 Appointify – Appointment Tracking System

Appointify is a desktop-based appointment creation and tracking system developed with Java and SQL.  
It is designed for businesses that operate with appointment systems such as clinics, private tutors, barber shops, etc.

---
### 👤 User Panel
- User registration with validation rules
- Secure login system
- View current appointments
- Cancel appointment
- Reschedule appointment
- Personalized dashboard

## 🧠 Validations

- Username must be unique (3–10 characters)
- Email validation
- Phone number validation
- Password validation

---

## 🔄 Appointment Logic

- If a time slot is occupied, it is marked with “X”
- Rescheduling cancels the old appointment and creates a new one
- Deleting a user automatically deletes their active appointments

---

## 🧩 Technologies Used

- Java (Swing GUI)
- MySQL
- JDBC
